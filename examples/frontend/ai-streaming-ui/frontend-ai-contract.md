# AI 前端交互契约

- 浏览器只请求自有后端，不持有模型供应商 API 密钥。
- 每次生成有稳定 `requestId`；旧请求事件不能修改新任务。
- 流式增量是临时投影，只有完成并经服务端确认的消息进入持久会话。
- “停止生成”同时中止浏览器读取并通知后端取消；二者都不等于供应商必然停止计费。
- 工具参数、结构化输出和生成式 UI 均在运行时校验，只渲染允许的组件。
- 写入或破坏性工具必须展示目标、参数、影响并取得明确确认。
- 模型文本按不可信输入处理，不直接使用 `innerHTML`，链接只允许安全协议。
- 错误、拒绝、限流、内容过滤、断网和部分完成拥有独立 UI 状态。
