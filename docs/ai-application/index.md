# 大模型应用开发

本专题关注如何把大模型能力转化为可靠的软件功能，而不是只编写几条 Prompt。

## 学习顺序

1. 模型 API、Token、流式输出和错误处理。
2. Prompt Engineering、Structured Output 和工具调用。
3. Embedding、文档切分、向量检索和重排序。
4. RAG 架构、引用、评估与幻觉治理。
5. Agent、工作流、MCP 和多智能体协作。
6. 成本、延迟、可观测性、安全和部署。

## 阶段项目

构建一个带来源引用的学习文档问答系统，再逐步加入评估集、工具调用和任务工作流。
