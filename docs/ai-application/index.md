# 大模型应用开发

本专题关注如何把大模型能力转化为可靠的软件功能，而不是只编写几条 Prompt。

## 课程导航

1. [大模型应用的完整请求链路：从用户输入到模型输出](./request-lifecycle.md)

后续课程将依次覆盖：

1. Token、上下文窗口与 Prompt 设计。
2. 结构化输出与工具调用。
3. Embedding、文档切分、向量检索和重排序。
4. RAG 架构、引用与上下文治理。
5. 评估、幻觉发现与质量回归。
6. Agent、确定性工作流与状态管理。
7. MCP 的客户端、服务端与安全边界。
8. 安全、成本、可观测性和部署。

每节课都会区分稳定的工程原则与快速变化的模型、SDK、协议细节。涉及外部服务的示例统一从环境变量读取密钥，不把凭据提交到仓库。

## 阶段项目

构建一个带来源引用的学习文档问答系统，再逐步加入评估集、工具调用和任务工作流。
