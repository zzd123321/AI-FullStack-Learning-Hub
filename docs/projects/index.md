# 项目实战

每个项目都服务于当前学习阶段，并尽量复用前一阶段成果。不会同时启动多个无法完成的大项目。

## 项目阶梯

| 项目 | 连接的知识 | 核心验收点 |
| --- | --- | --- |
| AI 全栈学习站 | VitePress、Markdown、Git、CI/CD | 内容结构清晰，可以搜索和部署 |
| 学习任务面板 | Vue 3、TypeScript、测试 | 类型完整，交互和异常状态可靠 |
| 用户与权限系统 | Java、Spring Boot、SQL、安全 | 权限模型正确，接口有测试 |
| 文档问答系统 | Python、Embedding、RAG、SSE | 回答有引用，可以量化评估 |
| AI 学习助手 | 全栈、Agent、MCP、工作流 | 任务可追踪，错误可恢复，成本可观察 |
