# 人工智能

这条路线的目标不是只会调用模型接口，而是建立足够的原理基础，能够理解模型能力、限制和评估方法。

## 学习顺序

1. 向量、矩阵、概率和微积分直觉。
2. NumPy、Pandas 和数据处理。
3. 监督学习、损失函数和模型评估。
4. 神经网络、反向传播和 PyTorch。
5. Attention、Transformer 和语言模型。
6. 训练、推理、Token、Embedding 与上下文窗口。

## 阶段项目

完成一个小型分类实验，并记录数据准备、训练、验证、指标和误差分析全过程。
