# 人工智能

这条路线的目标不是只会调用模型接口，而是建立足够的原理基础，能够理解模型能力、限制和评估方法。

## 学习顺序

1. **AI 基础概念**：先分清人工智能、机器学习、深度学习与生成式 AI。
2. **必要数学**：建立向量、矩阵、概率、导数和梯度的直觉。
3. **数据工具**：使用 NumPy、Pandas 表示、清洗和探索数据。
4. **机器学习**：理解监督学习、无监督学习、损失函数和模型评估。
5. **神经网络与深度学习**：理解多层表示、反向传播和优化。
6. **PyTorch**：用张量、自动微分和模块化组件实现训练流程。
7. **Transformer 与 NLP**：理解 Token、Embedding、Attention、Transformer 和语言模型。
8. **训练、推理与评估**：掌握数据划分、训练循环、推理过程、指标和误差分析。
