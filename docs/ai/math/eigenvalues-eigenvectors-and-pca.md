# 特征值、特征向量与主成分分析

> 本课讲稳定的数学原理，并用 Python 标准库实现二维 PCA。实际数值计算应使用经过验证的线性代数库。

## 1. 学习目标

完成本节后，你应该能够：

- 用几何语言解释特征向量和特征值。
- 说明为什么只有方阵才直接讨论特征分解。
- 理解实对称矩阵的特征分解为什么特别重要。
- 从中心化数据构造协方差矩阵。
- 推导 PCA 最大化投影方差的目标。
- 完整描述 PCA 的拟合、投影和重建流程。
- 解释主成分、解释方差与解释方差率。
- 说明 PCA 与 SVD 的关系以及实际实现为何常使用 SVD。
- 识别数据缩放、数据泄漏、异常值和符号不唯一等常见问题。

## 2. 哪些方向经过变换后没有转向

上一课把矩阵 \(\mathbf{A}\) 看作空间变换。大多数输入向量经过变换后，长度和方向都会改变。但某些特殊方向只会被拉伸、压缩或反向，不会偏离原来的直线。

若非零向量 \(\mathbf{v}\) 满足：

\[
\mathbf{A}\mathbf{v}=\lambda\mathbf{v}
\]

则：

- \(\mathbf{v}\) 是 \(\mathbf{A}\) 的**特征向量**；
- \(\lambda\) 是对应的**特征值**。

\(\lambda\) 描述沿该方向的缩放：

| 特征值 | 变换效果 |
| --- | --- |
| \(\lambda>1\) | 同方向拉伸 |
| \(0<\lambda<1\) | 同方向压缩 |
| \(\lambda=1\) | 保持不变 |
| \(\lambda=0\) | 压缩到零向量 |
| \(\lambda<0\) | 反向并按绝对值缩放 |

特征向量必须非零，因为零向量对任意 \(\lambda\) 都满足等式，无法定义一个特殊方向。

### 2.1 特征向量的长度不唯一

若 \(\mathbf{v}\) 是特征向量，则任意非零标量倍数 \(c\mathbf{v}\) 仍是同一特征值的特征向量：

\[
\mathbf{A}(c\mathbf{v})
=c\mathbf{A}\mathbf{v}
=c\lambda\mathbf{v}
=\lambda(c\mathbf{v})
\]

所以我们通常把特征向量归一化为单位长度。即便如此，\(\mathbf{v}\) 和 \(-\mathbf{v}\) 仍代表同一个特征方向。

## 3. 怎样求特征值

由：

\[
\mathbf{A}\mathbf{v}=\lambda\mathbf{v}
\]

得到：

\[
(\mathbf{A}-\lambda\mathbf{I})\mathbf{v}=\mathbf{0}
\]

要让这个齐次方程存在非零解，\(\mathbf{A}-\lambda\mathbf{I}\) 必须不可逆，因此：

\[
\det(\mathbf{A}-\lambda\mathbf{I})=0
\]

这称为特征方程。手算小矩阵时可以展开它；实际计算大型矩阵时不会简单照搬符号公式，而会使用更稳定高效的数值算法。

只有方阵才能直接写 \(\mathbf{A}\mathbf{v}=\lambda\mathbf{v}\)，因为输入和输出必须位于同一空间，才能比较是否保持方向。非方阵通常使用奇异值分解分析。

## 4. 特征空间、重根与不唯一性

对给定特征值 \(\lambda\)，所有满足：

\[
(\mathbf{A}-\lambda\mathbf{I})\mathbf{v}=0
\]

的向量连同零向量构成对应的**特征空间**。

同一特征值可能对应多个线性无关方向。例如恒等矩阵满足 \(\mathbf{I}\mathbf{v}=1\mathbf{v}\)，所有非零向量都是特征值 1 的特征向量。此时不存在唯一的“那一个”特征方向。

因此需要区分：

- 代数重数：特征值作为特征方程根出现的次数；
- 几何重数：对应特征空间的维数。

一般方阵不一定拥有足够多线性无关的特征向量，也不一定能在实数范围完成分解。

## 5. 对角化：在特征向量基下看变换

若 \(n\times n\) 矩阵拥有 \(n\) 个线性无关特征向量，把它们作为列组成：

\[
\mathbf{V}=
\begin{bmatrix}
\vert&&\vert\\
\mathbf{v}_1&\cdots&\mathbf{v}_n\\
\vert&&\vert
\end{bmatrix}
\]

并把特征值放入对角矩阵：

\[
\mathbf{\Lambda}=\operatorname{diag}(\lambda_1,\ldots,\lambda_n)
\]

则：

\[
\mathbf{A}=\mathbf{V}\mathbf{\Lambda}\mathbf{V}^{-1}
\]

这表示：

1. \(\mathbf{V}^{-1}\) 把向量换到特征向量坐标系；
2. \(\mathbf{\Lambda}\) 沿各特征方向独立缩放；
3. \(\mathbf{V}\) 把结果换回原坐标系。

对角矩阵没有坐标之间的混合，因此对角化能把复杂变换分解成若干独立方向上的简单缩放。

## 6. 实对称矩阵的特殊性质

若：

\[
\mathbf{A}=\mathbf{A}^{\mathsf T}
\]

则 \(\mathbf{A}\) 是实对称矩阵。它拥有非常好的性质：

- 所有特征值都是实数；
- 存在一组完整的标准正交特征向量；
- 不同特征值对应的特征向量彼此正交；
- 可以正交对角化：

  \[
  \mathbf{A}=\mathbf{Q}\mathbf{\Lambda}\mathbf{Q}^{\mathsf T}
  \]

  其中 \(\mathbf{Q}^{\mathsf T}\mathbf{Q}=\mathbf{I}\)。

协方差矩阵正是实对称矩阵，而且还是半正定矩阵，因此其特征值非负。这使 PCA 能够得到彼此正交、方差可排序的主方向。

## 7. 从数据走向协方差

设数据矩阵：

\[
\mathbf{X}\in\mathbb{R}^{n\times d}
\]

其中每行是一条样本，每列是一个特征。

### 7.1 先中心化

计算每个特征的均值：

\[
\boldsymbol{\mu}
=\frac{1}{n}\sum_{i=1}^{n}\mathbf{x}^{(i)}
\]

中心化数据：

\[
\mathbf{X}_c=\mathbf{X}-\boldsymbol{\mu}
\]

这里表示从每一行减去同一个均值向量。中心化让坐标原点移动到数据中心，使 PCA 分析“围绕均值的变化方向”，而不是原点位置。

### 7.2 协方差矩阵

样本协方差矩阵可写成：

\[
\mathbf{\Sigma}
=\frac{1}{n-1}\mathbf{X}_c^{\mathsf T}\mathbf{X}_c
\in\mathbb{R}^{d\times d}
\]

其中：

- 对角元素 \(\Sigma_{jj}\) 是第 \(j\) 个特征的样本方差；
- 非对角元素 \(\Sigma_{jk}\) 是两个特征的样本协方差；
- 正协方差表示倾向同向变化；
- 负协方差表示倾向反向变化；
- 接近 0 只表示线性共同变化较弱，不保证统计独立。

使用 \(1/n\) 会得到总体协方差形式。两者只相差统一缩放，不改变特征向量和解释方差率，但会改变特征值的绝对大小。

## 8. PCA 要寻找什么

PCA（Principal Component Analysis，主成分分析）寻找一个单位方向 \(\mathbf{w}\)，使中心化数据投影到该方向后的方差最大。

单条样本的投影坐标为：

\[
z=\mathbf{w}^{\mathsf T}(\mathbf{x}-\boldsymbol{\mu})
\]

所有投影的样本方差为：

\[
\operatorname{Var}(z)
=\mathbf{w}^{\mathsf T}\mathbf{\Sigma}\mathbf{w}
\]

因此第一主成分解决：

\[
\max_{\|\mathbf{w}\|_2=1}
\mathbf{w}^{\mathsf T}\mathbf{\Sigma}\mathbf{w}
\]

单位长度约束不可缺少，否则只要无限放大 \(\mathbf{w}\)，目标值就能无限增大。

这个优化问题的解是协方差矩阵最大特征值对应的单位特征向量，最大方差正是该特征值。

后续主成分在与已有主方向正交的约束下继续最大化方差，因此最终得到按特征值从大到小排列的标准正交方向。

## 9. PCA 完整流程

### 第一步：拟合均值

只使用训练数据计算 \(\boldsymbol{\mu}\)。

### 第二步：中心化训练数据

\[
\mathbf{X}_c=\mathbf{X}-\boldsymbol{\mu}
\]

### 第三步：寻找主方向

可对协方差矩阵做特征分解，或直接对中心化数据做 SVD。

### 第四步：选择前 \(k\) 个主成分

把它们按列组成：

\[
\mathbf{W}_k\in\mathbb{R}^{d\times k}
\]

### 第五步：投影降维

\[
\mathbf{Z}=\mathbf{X}_c\mathbf{W}_k
\in\mathbb{R}^{n\times k}
\]

### 第六步：近似重建

\[
\hat{\mathbf{X}}
=\mathbf{Z}\mathbf{W}_k^{\mathsf T}+\boldsymbol{\mu}
\]

若 \(k<d\)，重建通常有误差，因为被丢弃方向上的信息无法恢复。PCA 在正交线性投影中，以最小平方重建误差保留指定数量的维度。

## 10. 解释方差与解释方差率

第 \(j\) 个主成分对应的特征值 \(\lambda_j\) 是该方向上的方差。解释方差率为：

\[
r_j=\frac{\lambda_j}{\sum_{i=1}^{d}\lambda_i}
\]

前 \(k\) 个主成分的累计解释方差率：

\[
R_k=\frac{\sum_{j=1}^{k}\lambda_j}
{\sum_{i=1}^{d}\lambda_i}
\]

它回答“这 \(k\) 个线性方向保留了训练数据总方差的多少比例”，不等于：

- 保留了同等比例的语义信息；
- 下游任务准确率也保留同等比例；
- 被删除方向一定是噪声。

主成分数量应结合解释方差、重建误差、下游验证指标、计算成本和可解释性选择。

## 11. PCA 与 SVD 的关系

对中心化数据做奇异值分解：

\[
\mathbf{X}_c
=\mathbf{U}\mathbf{S}\mathbf{V}^{\mathsf T}
\]

则：

\[
\mathbf{\Sigma}
=\frac{1}{n-1}\mathbf{X}_c^{\mathsf T}\mathbf{X}_c
=\mathbf{V}\frac{\mathbf{S}^2}{n-1}\mathbf{V}^{\mathsf T}
\]

因此：

- \(\mathbf{V}\) 的列是 PCA 主方向；
- 奇异值 \(s_j\) 与特征值关系为 \(\lambda_j=s_j^2/(n-1)\)。

实际实现常直接对中心化数据使用 SVD，而不是显式构造协方差矩阵，因为形成 \(\mathbf{X}_c^{\mathsf T}\mathbf{X}_c\) 可能放大条件数问题，也可能增加计算或存储成本。具体算法选择取决于样本数、特征数、稀疏性和所需主成分数量。

SVD 适用于任意矩形矩阵；特征分解直接适用于方阵。二者相关但不是同一个概念。

## 12. 中心化不等于标准化

中心化只减去均值，不改变不同特征的量纲。若一个特征方差远大于其他特征，它可能主导 PCA。

是否在 PCA 前标准化为零均值、单位方差取决于语义：

- 不同单位、量级不可直接比较时，常考虑标准化；
- 原始方差本身有重要物理意义时，标准化可能丢失信息；
- 图像像素或同构传感器等同尺度特征未必需要逐列标准化。

scikit-learn 的 `PCA` 会中心化输入，但默认不会逐特征缩放。若需要标准化，应在训练管道中显式完成。

## 13. 数据泄漏：所有拟合步骤只看训练集

均值、标准差和 PCA 主方向都是从数据估计出的参数。正确流程是：

1. 划分训练集、验证集和测试集；
2. 只在训练集上拟合标准化参数与 PCA；
3. 用训练集得到的参数变换验证集和测试集；
4. 在变换后的数据上训练或评估后续模型。

如果先对全体数据拟合 PCA 再划分，测试集分布信息已经进入主方向，评估会产生数据泄漏。使用机器学习管道可以减少这种错误。

## 14. PCA 的重要性质与边界

### 14.1 PCA 是无监督线性方法

它不使用目标标签，只寻找总体方差最大的线性方向。高方差方向不一定最有利于分类，低方差方向也可能包含关键标签信息。

### 14.2 主成分是原特征的线性组合

PCA 通常不是直接挑选若干原始列，而是混合所有原特征生成新坐标。它属于特征提取，不等同于特征选择。

### 14.3 主成分符号不唯一

\(\mathbf{v}\) 与 \(-\mathbf{v}\) 表示同一主轴。不同库或运行环境可能返回相反符号，只要投影坐标同时反向，重建和解释方差不受影响。

### 14.4 相等或接近特征值会导致方向不稳定

若多个特征值相等，对应子空间中的正交基不唯一。特征值非常接近时，微小数据变化也可能显著旋转单个主成分，但它们共同张成的子空间仍可能稳定。

### 14.5 PCA 对尺度和异常值敏感

均值与协方差容易受极端值影响。实际使用前应检查数据质量，必要时采用稳健缩放、稳健 PCA 或其他方法。

### 14.6 PCA 只能发现线性结构

弯曲流形等非线性结构可能无法用少量线性主成分表示。Kernel PCA、自编码器等方法可以表达非线性关系，但引入了新的假设、成本和调参问题。

## 15. 白化不是普通 PCA 投影

普通 PCA 投影：

\[
\mathbf{Z}=\mathbf{X}_c\mathbf{W}_k
\]

各主成分彼此不相关，但方差分别为 \(\lambda_j\)。白化还会除以标准差：

\[
\mathbf{Z}_{white,j}
=\frac{\mathbf{Z}_j}{\sqrt{\lambda_j}}
\]

使保留分量的方差约为 1。白化会丢失相对尺度，并可能放大小特征值方向上的噪声；它不是“PCA 自动附带且总是更好”的步骤。

## 16. 特征值在 AI 中的其他位置

- 分析迭代系统的长期增长或衰减；
- 图拉普拉斯矩阵的谱方法与图聚类；
- Hessian 的局部曲率分析；
- 协方差、相关矩阵和核矩阵分析；
- 低秩近似、压缩和表示学习。

这些应用都利用“不变方向及其缩放强度”的思想，但不同矩阵的特征值含义不同，不能把 PCA 的“解释方差”解释套到所有矩阵上。

## 17. 可运行示例：手写二维 PCA

配套文件 `examples/ai/math/pca_from_scratch.py` 实现：

- 均值与中心化；
- 样本协方差矩阵；
- 幂迭代寻找最大特征值方向；
- 二维正交的第二方向；
- 主成分投影与重建；
- 解释方差率和平均平方重建误差。

运行：

```bash
python3 examples/ai/math/pca_from_scratch.py
```

幂迭代反复执行：

\[
\mathbf{v}_{t+1}
=\frac{\mathbf{A}\mathbf{v}_t}
{\|\mathbf{A}\mathbf{v}_t\|_2}
\]

在最大特征值绝对值唯一、初始向量在主方向上分量非零等条件下，它会收敛到主特征方向。这个教学实现只处理小型二维对称矩阵，不应替代通用数值库。

## 18. 使用 PCA 的检查清单

1. 数据行列语义是否明确？
2. 缺失值、异常值和类别特征是否已妥善处理？
3. 是否需要先做特征缩放？
4. 均值、缩放和 PCA 是否只在训练集拟合？
5. 主成分数量依据什么指标选择？
6. 下游任务是否真的受益，而不只是解释方差看起来很高？
7. 是否需要保存均值、缩放参数、主方向和版本信息以复现推理？
8. 数据分布变化后是否需要重新拟合和评估？

## 19. 常见误解

### “最大特征值就是矩阵中最大的元素”

不是。特征值描述整个线性变换沿特殊方向的缩放，不能从单个元素大小直接判断。

### “第一主成分是最重要的原始特征”

不是。第一主成分通常是多个原特征的线性组合，只是在训练数据上具有最大方差。

### “解释方差率 95% 就保留了 95% 语义”

不是。它只表示保留了训练数据总方差的 95%，不直接衡量语义或预测能力。

### “PCA 前必须总是标准化”

不是。是否缩放取决于单位、数据生成过程和任务语义，但必须明确自己的选择。

### “特征向量符号不同说明计算错了”

不是。单位特征向量整体乘以 -1 仍表示同一方向。

### “对全数据先做 PCA 没关系，因为它没用标签”

错误。PCA 使用了测试数据的特征分布，仍会产生数据泄漏。

## 20. 稳定原理、版本行为与工程实践

| 类型 | 本课示例 | 应该如何对待 |
| --- | --- | --- |
| 稳定原理 | 特征方程、对称矩阵性质、PCA 最大方差目标 | 长期掌握 |
| 版本行为 | PCA 求解器选择、返回排序、API 默认值 | 查对应版本文档 |
| 工程实践 | 缩放、主成分数、批量或增量算法 | 用数据规模和验证指标决定 |

例如 NumPy 面向实对称矩阵提供 `eigh`，返回的特征值顺序属于 API 行为；数学上特征值集合本身没有“必须按升序返回”的要求。

## 21. 本课小结

- 特征向量是矩阵变换下不改变所在直线的非零方向，特征值描述该方向的缩放。
- 可对角化矩阵在特征向量基下等价于沿各方向独立缩放。
- 实对称矩阵拥有实特征值和标准正交特征向量；协方差矩阵还具有非负特征值。
- PCA 对中心化数据寻找方差最大的正交方向，主方向是协方差矩阵的特征向量。
- 特征值给出各主成分方差，解释方差率只描述方差保留比例。
- PCA 投影是线性降维，重建会在丢弃方向上产生信息损失。
- 实际实现常直接对中心化数据做 SVD，而不显式构造协方差矩阵。
- 中心化不等于标准化；所有拟合步骤必须只使用训练数据。
- 主成分符号不唯一，接近或重复特征值还会导致单个方向不稳定。
