# 向量长度、距离、正交与余弦相似度

> 本课讨论稳定的数学原理。示例使用 Python 标准库；NumPy、PyTorch 和向量数据库的具体 API 将在对应模块按版本讲解。

## 1. 学习目标

完成本节后，你应该能够：

- 区分范数、距离、点积和相似度。
- 计算并解释 \(L^1\)、\(L^2\) 和 \(L^\infty\) 范数。
- 说明一个函数要成为范数或距离度量需要满足哪些条件。
- 使用点积理解夹角、正交和向量投影。
- 正确计算余弦相似度，并处理零向量问题。
- 解释归一化后点积、余弦相似度与欧氏距离之间的关系。
- 根据数据语义选择距离或相似度，而不是机械套用公式。
- 说明 Embedding 检索的余弦相似度与 Attention 缩放点积的区别。

## 2. 从点积出发建立几何

上一课从代数角度理解了点积：对应位置相乘后求和。

\[
\mathbf{x}^{\mathsf T}\mathbf{y}
=\sum_{i=1}^{d}x_i y_i
\]

点积还有一个几何表达。对两个非零向量：

\[
\mathbf{x}^{\mathsf T}\mathbf{y}
=\|\mathbf{x}\|_2\|\mathbf{y}\|_2\cos\theta
\]

这里：

- \(\|\mathbf{x}\|_2\) 和 \(\|\mathbf{y}\|_2\) 是向量长度；
- \(\theta\) 是二者夹角；
- \(\cos\theta\) 描述方向的一致程度。

这条公式把本课概念连在一起：

```text
点积
├── 自己与自己点积 → 长度
├── 两向量点积为 0 → 正交
├── 除以两个长度 → 余弦相似度
└── 用点积提取某方向分量 → 投影
```

## 3. 范数：怎样定义向量的“大小”

范数（norm）把向量映射为一个非负标量：

\[
\|\cdot\|:V\rightarrow\mathbb{R}_{\ge0}
\]

一个函数要成为范数，必须对所有向量 \(\mathbf{x},\mathbf{y}\) 和标量 \(c\) 满足：

1. **非负与确定性**

   \[
   \|\mathbf{x}\|\ge0,
   \qquad
   \|\mathbf{x}\|=0\iff\mathbf{x}=\mathbf{0}
   \]

2. **绝对齐次性**

   \[
   \|c\mathbf{x}\|=|c|\|\mathbf{x}\|
   \]

3. **三角不等式**

   \[
   \|\mathbf{x}+\mathbf{y}\|
   \le\|\mathbf{x}\|+\|\mathbf{y}\|
   \]

这些条件刻画了“长度”应该有的行为：长度不能为负；只有零向量长度为零；缩放几倍，长度就缩放几倍；直接走到终点不会比绕路更长。

## 4. \(L^p\) 范数

对 \(\mathbf{x}\in\mathbb{R}^{d}\) 和 \(p\ge1\)，\(L^p\) 范数定义为：

\[
\|\mathbf{x}\|_p
=
\left(\sum_{i=1}^{d}|x_i|^p\right)^{1/p}
\]

### 4.1 \(L^1\) 范数：绝对值之和

\[
\|\mathbf{x}\|_1=\sum_i|x_i|
\]

例如：

\[
\left\|
\begin{bmatrix}3\\-4\end{bmatrix}
\right\|_1
=|3|+|-4|=7
\]

它又称曼哈顿范数，因为在只能沿水平与垂直街道移动的网格中，路径长度像各坐标差的绝对值之和。

机器学习中，\(L^1\) 常与稀疏性相关：在优化目标中加入 \(L^1\) 惩罚，往往更容易让某些参数精确变为 0。但这不是“用了 \(L^1\) 就必然稀疏”的无条件结论，还取决于目标函数、优化过程和正则化强度。

### 4.2 \(L^2\) 范数：欧氏长度

\[
\|\mathbf{x}\|_2
=\sqrt{\sum_i x_i^2}
=\sqrt{\mathbf{x}^{\mathsf T}\mathbf{x}}
\]

对 \([3,-4]^{\mathsf T}\)：

\[
\|\mathbf{x}\|_2=\sqrt{3^2+(-4)^2}=5
\]

它就是熟悉的直线距离所使用的长度，也是机器学习中最常见的范数之一。

### 4.3 \(L^\infty\) 范数：最大绝对坐标

\[
\|\mathbf{x}\|_\infty=\max_i|x_i|
\]

因此：

\[
\left\|
\begin{bmatrix}3\\-4\end{bmatrix}
\right\|_\infty=4
\]

它只关心绝对值最大的坐标，可用于表达“任何单个坐标的变化都不能超过某个上限”。

### 4.4 三种范数关注点不同

| 范数 | 计算方式 | 直觉重点 |
| --- | --- | --- |
| \(L^1\) | 绝对值之和 | 所有坐标变化的总量 |
| \(L^2\) | 平方和再开方 | 几何直线长度 |
| \(L^\infty\) | 最大绝对值 | 最大单坐标变化 |

它们不是谁“更正确”，而是在用不同方式定义大小。

## 5. 两个看起来像范数、其实不是范数的量

### 5.1 “\(L^0\) 范数”

实践中有时把非零元素数量记作 \(\|\mathbf{x}\|_0\)：

\[
\|\mathbf{x}\|_0=\#\{i:x_i\ne0\}
\]

但它不满足绝对齐次性。例如非零标量 \(c\) 不改变非零元素数量，因此：

\[
\|c\mathbf{x}\|_0=\|\mathbf{x}\|_0
\]

一般并不等于 \(|c|\|\mathbf{x}\|_0\)。所以“\(L^0\) 范数”只是常用俗称，不是严格范数。

### 5.2 平方 \(L^2\) 范数

\[
\|\mathbf{x}\|_2^2=\sum_i x_i^2
\]

平方形式在优化中非常常见，因为没有最外层平方根且容易求导。但它也不是范数：

\[
\|c\mathbf{x}\|_2^2=c^2\|\mathbf{x}\|_2^2
\]

而范数要求缩放因子是 \(|c|\)。使用它没有问题，关键是不要把名称和性质混淆。

## 6. 矩阵也可以有范数

矩阵范数有多种定义。深度学习中常见的 Frobenius 范数为：

\[
\|\mathbf{A}\|_F
=\sqrt{\sum_{i,j}A_{ij}^2}
\]

它可以理解为先把矩阵所有元素排成一个长向量，再计算 \(L^2\) 范数。

还有谱范数、核范数等，它们度量矩阵的不同性质，不能一概而论。后续正则化和矩阵分解课程会在需要时引入。

## 7. 距离：衡量两个对象相隔多远

距离度量（metric）是一个函数：

\[
d:X\times X\rightarrow\mathbb{R}_{\ge0}
\]

对任意 \(\mathbf{x},\mathbf{y},\mathbf{z}\)，它需要满足：

1. 非负性：\(d(\mathbf{x},\mathbf{y})\ge0\)；
2. 同一性：\(d(\mathbf{x},\mathbf{y})=0\iff\mathbf{x}=\mathbf{y}\)；
3. 对称性：\(d(\mathbf{x},\mathbf{y})=d(\mathbf{y},\mathbf{x})\)；
4. 三角不等式：

   \[
   d(\mathbf{x},\mathbf{z})
   \le d(\mathbf{x},\mathbf{y})+d(\mathbf{y},\mathbf{z})
   \]

任意范数都能诱导一个距离：

\[
d_p(\mathbf{x},\mathbf{y})
=\|\mathbf{x}-\mathbf{y}\|_p
\]

因此：

- 曼哈顿距离：\(\|\mathbf{x}-\mathbf{y}\|_1\)；
- 欧氏距离：\(\|\mathbf{x}-\mathbf{y}\|_2\)；
- Chebyshev 距离：\(\|\mathbf{x}-\mathbf{y}\|_\infty\)。

### 7.1 平方欧氏距离不是距离度量

平方欧氏距离：

\[
d^2(\mathbf{x},\mathbf{y})
=\|\mathbf{x}-\mathbf{y}\|_2^2
\]

经常作为损失函数使用，也与欧氏距离产生相同的最近邻排序，因为平方函数在非负区间单调递增。但它一般不满足三角不等式，因此严格说不是 metric。

例如实数轴上的 0、1、2：

\[
d^2(0,2)=4
>d^2(0,1)+d^2(1,2)=2
\]

## 8. 特征尺度会改变距离

假设样本包含：

```text
[年龄, 年收入]
```

年龄可能在几十的量级，年收入可能在几十万的量级。直接使用欧氏距离时，收入坐标很可能主导结果，并不代表它在任务中一定更重要。

因此，计算距离前常需要：

- 统一单位；
- 标准化或归一化特征；
- 根据业务意义设置权重；
- 学习适合任务的表示或度量。

“公式算得出来”只代表数值上可执行，不代表各坐标具有可比较的语义。

## 9. 夹角与余弦

由点积几何公式，对非零向量有：

\[
\cos\theta
=
\frac{\mathbf{x}^{\mathsf T}\mathbf{y}}
{\|\mathbf{x}\|_2\|\mathbf{y}\|_2}
\]

根据 Cauchy–Schwarz 不等式：

\[
|\mathbf{x}^{\mathsf T}\mathbf{y}|
\le\|\mathbf{x}\|_2\|\mathbf{y}\|_2
\]

所以余弦值位于 \([-1,1]\)：

| 余弦值 | 方向关系 |
| --- | --- |
| 接近 1 | 方向接近一致 |
| 接近 0 | 接近正交 |
| 接近 -1 | 方向接近相反 |

这只描述方向，不自动等同于“语义相似”。只有当模型和训练目标让向量方向承载了相关语义时，余弦值才有对应的任务意义。

### 9.1 零向量没有方向

若任一向量为零向量，分母为 0，余弦相似度在数学上未定义。不同软件可能选择报错、加入很小的 \(\varepsilon\)、返回特定值或产生 `NaN`。

工程上加入 \(\varepsilon\) 是为了数值稳定，不是把“零向量与其他向量的真实夹角”定义出来。调用库函数时要检查其零范数处理规则。

## 10. 正交与标准正交

若：

\[
\mathbf{x}^{\mathsf T}\mathbf{y}=0
\]

则称 \(\mathbf{x}\) 与 \(\mathbf{y}\) **正交**。当两个向量都非零时，它们的夹角为 \(90^\circ\)。零向量与每个向量点积都为 0，因此代数上与所有向量正交，但零向量没有方向，也不能成为基向量。

如果一组向量两两正交且每个长度都为 1，它们是**标准正交**（orthonormal）的。

标准正交基特别方便。若 \(\mathbf{q}_1,\ldots,\mathbf{q}_n\) 是标准正交基，则向量坐标直接由点积得到：

\[
c_i=\mathbf{q}_i^{\mathsf T}\mathbf{x}
\]

若方阵 \(\mathbf{Q}\) 的列是标准正交向量，则：

\[
\mathbf{Q}^{\mathsf T}\mathbf{Q}=\mathbf{I},
\qquad
\mathbf{Q}^{-1}=\mathbf{Q}^{\mathsf T}
\]

这样的正交矩阵保持点积、长度、夹角和距离。旋转与镜像是典型例子。

## 11. 投影：取出沿某个方向的分量

把 \(\mathbf{x}\) 投影到非零向量 \(\mathbf{v}\) 所在直线上：

\[
\operatorname{proj}_{\mathbf{v}}(\mathbf{x})
=
\frac{\mathbf{x}^{\mathsf T}\mathbf{v}}
{\mathbf{v}^{\mathsf T}\mathbf{v}}\mathbf{v}
\]

若 \(\mathbf{v}\) 已经是单位向量 \(\mathbf{q}\)，公式简化为：

\[
\operatorname{proj}_{\mathbf{q}}(\mathbf{x})
=(\mathbf{x}^{\mathsf T}\mathbf{q})\mathbf{q}
\]

于是向量可以分解为：

\[
\mathbf{x}
=\operatorname{proj}_{\mathbf{v}}(\mathbf{x})
+\mathbf{r}
\]

残差：

\[
\mathbf{r}
=\mathbf{x}-\operatorname{proj}_{\mathbf{v}}(\mathbf{x})
\]

与 \(\mathbf{v}\) 正交。投影是最小二乘、线性回归、PCA 和正交化的重要基础。

## 12. 归一化：把向量放到单位球面上

对非零向量进行 \(L^2\) 归一化：

\[
\hat{\mathbf{x}}
=\frac{\mathbf{x}}{\|\mathbf{x}\|_2}
\]

归一化后：

\[
\|\hat{\mathbf{x}}\|_2=1
\]

因此两个归一化向量的点积就是余弦相似度：

\[
\hat{\mathbf{x}}^{\mathsf T}\hat{\mathbf{y}}
=\cos\theta
\]

它们的欧氏距离还满足：

\[
\|\hat{\mathbf{x}}-\hat{\mathbf{y}}\|_2^2
=2-2\cos\theta
\]

所以当所有向量都经过 \(L^2\) 归一化后：

- 最大化点积；
- 最大化余弦相似度；
- 最小化欧氏距离；

会得到相同的排序。这个等价关系依赖“所有向量都已归一化”这一条件。

## 13. 余弦相似度与“余弦距离”

余弦相似度定义为：

\[
s_{cos}(\mathbf{x},\mathbf{y})
=\frac{\mathbf{x}^{\mathsf T}\mathbf{y}}
{\|\mathbf{x}\|_2\|\mathbf{y}\|_2}
\]

许多库把：

\[
d_{cos}=1-s_{cos}
\]

称为 cosine distance。它是常用的不相似度，但在一般向量空间上不保证满足三角不等式，因此不要仅凭名字假设它是严格的数学 metric。

如果确实需要单位球面上的角度度量，可以考虑：

\[
d_{angle}(\mathbf{x},\mathbf{y})
=\arccos(s_{cos}(\mathbf{x},\mathbf{y}))
\]

但具体系统选择什么函数，还要考虑排序目标、计算成本、索引实现和零向量策略。

## 14. 欧氏距离、点积和余弦怎样选择

### 欧氏距离

同时关注方向与模长，适合坐标差本身有意义且尺度已经合理处理的场景。

### 点积

同时受夹角和两个向量模长影响：

\[
\mathbf{x}^{\mathsf T}\mathbf{y}
=\|\mathbf{x}\|_2\|\mathbf{y}\|_2\cos\theta
\]

模长可能承载置信度、流行度或强度，也可能只是训练过程留下的偏差。是否保留它必须由模型训练方式和任务决定。

### 余弦相似度

消除整体正比例缩放，只比较方向。若 \(\mathbf{y}=c\mathbf{x}\) 且 \(c>0\)，余弦相似度为 1，无论 \(c\) 多大。

没有一种度量对所有 Embedding 都最好。应优先遵循模型发布者的训练目标和推荐度量，并用真实验证集评估召回质量。

## 15. AI 中的典型应用

### 15.1 最近邻分类

k 近邻算法根据某种距离找出最接近的训练样本，再用邻居标签进行预测。结果高度依赖：

- 特征表示；
- 特征缩放；
- 距离函数；
- \(k\) 的选择；
- 高维数据的分布。

### 15.2 Embedding 检索

文本、图像或用户经过模型编码为向量后，可以按余弦相似度、点积或欧氏距离检索邻居。

但 Embedding 相近只说明**在该模型、该训练数据和该度量下相近**，不保证事实等价、因果相关或在所有业务语境中相似。检索系统仍需要任务评估、权限过滤和结果校验。

### 15.3 高维空间中的距离

在某些高维数据分布下，最近与最远距离的相对差异会变小，距离可能出现集中现象。这是“维度灾难”的一部分，但不是所有高维表示都必然失效。

实际效果取决于数据是否落在较低维结构上、模型是否学到有意义表示、采用何种度量，以及索引近似误差。不能只根据向量维数判断检索质量。

### 15.4 Attention 的缩放点积不是余弦相似度

Transformer 的缩放点积 Attention 使用：

\[
\operatorname{score}(\mathbf{q},\mathbf{k})
=\frac{\mathbf{q}^{\mathsf T}\mathbf{k}}{\sqrt{d_k}}
\]

它除以的是维数相关常数 \(\sqrt{d_k}\)，而不是 \(\|\mathbf{q}\|_2\|\mathbf{k}\|_2\)。因此它仍然保留 Query 和 Key 模长的影响，不等于余弦相似度。

原始 Transformer 论文给出的动机是：在分量独立、均值为 0、方差为 1 的简化假设下，点积方差随 \(d_k\) 增长；除以 \(\sqrt{d_k}\) 可以控制尺度，减少 softmax 进入梯度很小的饱和区域。这个解释依赖统计假设，应理解为设计动机而不是所有真实激活都严格满足的定律。

## 16. 数值计算中的边界问题

### 16.1 浮点数不应总用精确相等

理论上正交要求点积等于 0，但浮点计算常得到非常接近 0 的小数。工程中通常使用绝对误差与相对误差容限判断“足够接近”。

### 16.2 余弦值可能轻微越界

由于舍入误差，计算结果可能得到 `1.0000000002`。在传给 `acos` 前，通常需要把值限制到 `[-1, 1]`。

### 16.3 很小的范数

即使向量不是精确零向量，除以极小范数也会放大噪声。库函数常引入 `eps` 下限。`eps` 的取值与数据类型和数值范围有关，不应脱离上下文复制。

### 16.4 大规模检索

逐个扫描所有向量能得到精确结果，但数据量大时成本高。工程系统常使用近似最近邻索引，在召回率、延迟、内存与索引更新时间之间权衡。

```python
def cosine_similarity(left: Vector, right: Vector) -> float:
    left_norm = l2_norm(left)
    right_norm = l2_norm(right)
    if left_norm <= 1e-12 or right_norm <= 1e-12:
```

## 18. 选择度量的检查清单

面对一个相似度问题时，依次确认：

1. 每个坐标是否具有可比较的单位和尺度？
2. 向量模长是否包含任务需要的信息？
3. 模型训练时优化或推荐的是哪种度量？
4. 数据中是否可能出现零向量或极小范数？
5. 系统需要严格 metric，还是只需要稳定的排序分数？
6. 是否已经对查询向量和候选向量采用一致的归一化？
7. 离线验证集上的召回、排序与业务指标如何？
8. 精确检索的成本是否可接受，是否需要近似索引？

数学公式决定一个分数怎样计算，验证数据才决定它是否适合任务。

## 19. 常见误解

### “距离小就表示语义一定相同”

不是。距离只度量给定表示空间中的数值关系，语义由模型、数据和训练目标共同决定。

### “余弦相似度为 0 就表示完全无关”

它只表示两个非零向量正交。在学习到的表示中，正交是否等于业务上的“无关”需要实证验证。

### “余弦相似度一定在 0 到 1 之间”

普通实数向量的余弦相似度范围是 \([-1,1]\)。只有在向量坐标或模型结构施加额外限制时，范围才可能缩小。

### “归一化不会损失信息”

归一化会丢弃原始模长。若模长包含置信度或强度信息，这可能影响任务效果。

### “平方欧氏距离和欧氏距离性质完全相同”

它们对非负距离产生相同排序，但平方欧氏距离通常不满足三角不等式，数值尺度和梯度也不同。

### “Attention 就是在算余弦相似度”

标准缩放点积 Attention 不会除以 Query、Key 的范数，因此不是余弦相似度。

## 21. 本课小结

- 范数衡量单个向量的大小，距离衡量两个对象的间隔，相似度衡量某种接近程度。
- \(L^1\)、\(L^2\) 和 \(L^\infty\) 对“大小”有不同偏好；“\(L^0\)”和平方 \(L^2\) 严格说不是范数。
- 任意范数都能通过 \(d(\mathbf{x},\mathbf{y})=\|\mathbf{x}-\mathbf{y}\|\) 诱导距离。
- 平方欧氏距离常用于优化，但一般不是严格距离度量。
- 点积连接向量长度和夹角；点积为 0 表示正交。
- 投影提取沿目标方向的分量，残差与目标方向正交。
- 余弦相似度只对非零向量定义，比较方向并忽略整体正比例缩放。
- 所有向量归一化后，点积、余弦相似度和欧氏距离产生等价排序。
- Embedding 的度量应匹配模型训练方式并经过任务验证；Attention 缩放点积不等于余弦相似度。

下一课将进入矩阵分解的关键直觉：特征值、特征向量与主成分分析，理解模型和数据中“重要方向”如何被发现。
